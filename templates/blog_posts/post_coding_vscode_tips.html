by {{ post.author }}
  • ALT

    Multi-cursor editing

    alt - shifts highlighted line (up or down)
    ctrl + alt - Add another cursor (up or down)
    ctrl + Shift + L - Highlights all occurances of the word

  • copy

    Line Actions

    Shift + alt (up/down) - Copies a line and inserts it (above/below)
    ctrl + shift + K - Deletes entire line
    ctrl + / - Comments out entire line

  • Rename refactoring

    Rename refactoring

    Hover over class name or variable name, press F2,
    to rename the object everywhere in the project.

  • Folder code

    Folder code

    Ctrl + shift + [ - Fold the code
    Ctrl + shift + ] - Unfold the code
    Ctrl + K ctrl + 0 - Fold all code
    Ctrl + K ctrl + J - Unfold all code

  • Errors

    Errors

    F8 - cycles through all errors (cautions) in the project
